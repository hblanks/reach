About Reach

Reach aims to make it easier to track how scientific research influences policy and to analyse trends in the use of scientific research in policy.

Open Source

Reach's code is entirely open sources! You can check it out on our Github Repository.

Accuracy

Reach uses software and machine learning to process thousands of policy documents. Machine learning is not 100% accurate as it is based on probability. We are constantly working on Reach to improve its accuracy.

Transparency

We are committed to supporting responsible data innovation and product development. If you are interested in following our work, we publish regular blogs on our Medium channel.

Learn more about Reach

How Reach works

Reach searches through thousands of policy documents, extracts the references, and checks them against scientific publications.

1. Download

Reach downloads PDF from a number of organisation websites

2. Extract

Using machine learning, the references from the PDF are extracted

3. Match

The references are checked against the EPMC database of scientific publications
