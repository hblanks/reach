About Reach

Reach aims to make it easier to track how scientific research influences policy and to analyse trends in the use of scientific research in policy.

Open Source

Reach's code is entirely open sources! You can check it out on our Github Repository.

Accuracy

Reach uses software and machine learning to process thousands of policy documents. Machine learning is not 100% accurate as it is based on probability. We are constantly working on Reach to improve its accuracy.
